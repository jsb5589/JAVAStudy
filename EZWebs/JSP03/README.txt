[액션 태그]

1. 인클루드 액션 태그(include)
   - JSP를 포함
   - <jsp:include>

2. 포워드 액션 태그
   - ReqeustDispatcher 포워딩
   - <jsp:forward>
   
3. 유즈빈 액션 태그
   - 자바의 new 연산자를 대신하는 태그
   - 인자 없는 생성자를 가져야 한다.(디폴트 생성자)
   - <jsp:useBean>
   
4. Set 프로퍼티 액션 태그
   - 자바의 setter
   - 첫 글자는 소문자 
   - <jsp:setProperty>
   
5. Get 프로퍼티 액션 태그
   - 자바의 getter
   - 첫 글자는 소문자 
   - <jsp:getProperty>
