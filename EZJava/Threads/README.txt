[멀티 스레드]

1. 프로세서(Processor)
	- CPU
	- 물리적인 하드웨어 장치
	
2. 프로세스(Process)
	- 하나의 애플리케이션
	- 물리적인 실행 모듈 단위

3. 멀티 테스크(Multi-Tasking)
	- 다중 처리
	- 시분할 처리
		-> CPU가 놀고 있는 시간에 다른 작업을 처리
		-> 시간을 나눠서 여러 작업을 동시에 처리
	- 개념적인 작업 단위

4. 스레드(Thread)
	- 작업단위(Task)
	- 프로세스에 종속된 작업 단위
	- 하나의 프로세스 안에 다중으 스레드들은 자원을 공유
	- 하나의 프로세스는 최소 하나의 스레드를 가지고 있다.
	- 메인스레드: 프로세스가 처음 실행되면 메인으로 처리하는 스레드
	- 작업스레드: 필요에의해서 생성하여 처리하는 스레드
	- 작업우선순위
	- 동기화: 작업순서, 협업
	- 자료의 공유: 접근, 보호
	- 자원의 할당
