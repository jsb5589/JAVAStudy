컬렉션(Collection)

1. List: 배열, 순서를 유지하고 저장, 중복저장가능, 순차적인 연속된 데이터 처리
   - ArrayList
   - Vector : 쓰레드 안전(Thread Safe)
   - LinkedList
    
2. Set: 집합, 순서를 유지하지 않고 저장, 중복 저장 안 됨, 합집합, 교집합
   - HashSet
   - TreeSet
   
3. Map: 사전, 키와 값의 쌍으로 저장, 키는 중복 저장 안됨, 딕셔너리, 비순차 탐색
   - HashMap
   - Hashable : 쓰레드 안전(Thread Safe)
   - TreeMap
   - Properties
